Java高级开发工程师

岗位职责

1. 参与电商类SAAS平台的产品研发和管理工作;
2. 为公司服务团队和客户提供产品实施的技术顾问服务;
3. 领导技术团队完成项目任务，参与关键客户的产品实施，确保实施方案的正确和成功。

职位要求

1. 大专及本科以上学历，计算机专业；具备至少3年以上软件开发开发工作经验，至少1年以上技术团队管理经验;
2. 有领导或参与大中型软件项目开发的经验，能独立或指导他人完成软件系统分析及设计工作;
3. 熟悉Java应用主流前后端开发技术，有丰富的开发经验;
4. 精通Java语言编程，熟练使用Java开发工具，熟悉J2EE架构和开发模式，熟悉Oracle、MySQL等数据库系统;
5. 熟悉以下技术优先：Redis、Memcached等缓存技术；ElasticSearch、Solr、Lucene等搜索技术；AngularJS、Ionic等JavaScript框架；发起或参与过Java或JavaScript开源项目;
6. 熟悉项目实施从售前、需求、设计、开发、测试、部署到运维的全流程，有互联网商务项目经验优先;
7. 较好的沟通能力、协调能力、书面表达能力，具备良好的团队协作能力，能承受一定工作压力;思路清晰，善于思考，能独立分析和解决问题；具有创新思维，能不断推陈置新。

软件销售

岗位职责

1. 利用电话、上门拜访等多元化的方式挖掘潜在客户;
2. 根据客户的不同需求，有针对性的为客户设计及推销合适的产品和服务;
3. 每月完成既定的销售指标（包括客户开发与维护、销售回款等）;
4. 协助经理完成销售部其他相关事宜

职位要求

1. 大专以上学历，有商务谈判、电话销售、网络销售及面销等销售经验优先录取;
2. 热爱销售岗位，富有挑战精神，善用网络资源与各种人脉渠道;
3. 有良好的团队精神，能承受工作压力;有良好的沟通能力和亲和力;具有较强竞争意识，思维敏捷，善于开拓创新。

测试工程师

岗位职责

1. 根据测试标准和规范进行软件测试，能够保质保量按时完成测试任务;
2. 执行功能测试、回归测试;
3. 软件可靠性和安全性设计检查;
4. 设计测试用例，对测试中发现的问题进行及时跟踪和反馈，分析和解决问题;
5. 协助主管完善测试标准和规范。

职位要求

1. 大专以上学历，计算机类专业;
2. 二年软件测试经验，有APP软件测试经验，熟悉黑盒或白盒测试；了解Oracle数据库，熟练使用SQL语句进行增删改查等;
3. 熟悉一门以上的编程语言；熟悉相关测试工具(QTP、VS、LR等)；掌握计算机操作系统、网络知识;
4. 具备良好的逻辑思维能力和自学能力，富有责任心和团队合作精神；工作细致耐心;
5. 具有计算机软件维护能力，具有较强的解决问题能力。

项目管理

岗位职责

1. 协调短周期项目进度和需求计划;
2. 参与项目进度、范围和质量的跟踪和服务实施，确保项目现场的系统稳定运行;
3. 协调客户和项目组开发人员的沟通;
4. 帮助提供解决方案建议，将公司产品进行组装和配置调整，适应不同客户需求。

职位要求

1. 大专以上，计算机、软件工程、电子信息、网络工程等相关专业;
2．一年以上软件实施，技术支持工作经验，时尚服饰行业经验优先;
3．具备一定的ms-sql/mysql ，了解project、visio管理工具，了解oracle数据库优先;
4．熟悉网络设备CISCO、H3C、JUNIPER、HUAWEI路由器、交换机、AP等设备的调试;
5．熟悉服务器搭建、配置、优化，包括WINDOWS、LINUX系统操作;
6．具有团队精神，乐于合作；性格外向，乐于沟通，积极肯干，能适当出差。
